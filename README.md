# cns-worldcup26-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态页面资源的集中存放与展示用途。

## 项目简介

本仓库主要存放与归档一系列独立的 HTML 页面，这些页面可被用于信息展示、参考或索引。所有页面均为自包含的静态文件，不依赖外部动态服务或数据库。

## 目录说明

```
.
├── index.html          # 仓库入口页或导航页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心页面存放目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面引用的公共资源文件，按类型分类存放。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接通过浏览器打开。
- 页面之间无强依赖关系，可单独使用或通过索引页跳转。
- 页面内容可能涉及特定主题或信息集合，但本仓库不为其内容做任何背书。
- 归档的页面可能随时间更新或调整，请以仓库中的最新版本为准。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 用浏览器直接打开 `pages/` 目录下的任意 HTML 文件，或打开根目录的 `index.html` 查看页面索引。
3. 也可将本仓库部署到任何静态托管服务，即可通过对应 URL 访问页面。
